Alarme com sensor de presença e controle por app mobile.

Embarcado construído com Arduino ESP32 e app mobile desenvolvido em react-native.

Componentes para reprodução do embarcado:
* Placa DOIT ESP32
* PIR Motion
* Passive Buzzer
* Dois LEDs e resistores
